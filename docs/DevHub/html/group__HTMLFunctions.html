Creates a Nh_HTML_Document structure required for further computing.

Parameters
location_pThe location of the HTML document specified by a set of chars. This will be parsed by Nh_createURI.
Tab_pThe tab that will contain the document.
Returns
NH_SUCCESS on success, various error codes on failure.
